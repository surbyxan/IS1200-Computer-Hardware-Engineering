This Repository contains the labwork and the project done for the course IS1200 at KTH
The course labs and project was done on a chipkit with a basic I/O shield, with a 
tool chain also supplied by the course. 

Lab1 is done i MIPS assembly 

Lab 2-3 is done in C

The project is developed to run on the chip kit and is an interpretation of the 
classic game Flappy Bird. The project is done in C.
